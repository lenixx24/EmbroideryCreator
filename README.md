# Інструкція користувача

На початку програми користувач потрапляє в головне меню з трьома основними опціями:

- **Завантажити з файлу** — відкриває провідник, де можна обрати лише файли з розширенням .png

- **Створити новий проєкт** — відкриває вікно з можливістю вибору ширини та висоти полотна або завантаження стандартного полотна з іменем

- **Вийти** — закриває вікно програми


Після підтвердження вибору відкриється основна програма з вишивкою імені Ангеліна, з фото з файлу або порожнім полотном. Програма поділяється на три основні зони: верхня, полотно та нижня.


### Верхня панель

В верхній панелі є кнопка повернення в головне меню та поле для вводу імені файлу, в який користувач бажає зберегти вишивку.

> Якщо в назві використано заборонені символи або вона порожня - з’явиться вікно з повідомленням про це.

> Якщо ім’я файлу правильне, вишивка збережеться в папці saves у форматі png.


### Полотно

Полотно по середині екрану безпосередньо містить вишивку. Натискання мишкою в межах вікна буде малювати/стирати обрані стібки.

### Нижня панель

Нижня панель - містить керування редагуванням вишивки.


- **Очищення:** Для очищення полотна треба натиснути кнопку “Очистити” та підтвердити вибір в спливаючому вікні.

- **Гумка:** Перемикач “Гумка” встановлює режим гумки - всі натиснуті клітинки очищаються.

- **Сітка:** “Сітка” відповідає за відображення сітки на полотні та в збереженому файлі.

- **Колір:** Натискання на “Колір” дає можливість обрати колір вишивання в палітрі кольорів.

- **Симетрія:** Режими симетрії перемикаються в списку “Симетрія”, якщо обрати якийсь режим - кожен намальований стібок буде відображатись симетрично по обраній осі. Зображення трикутників відображають полотно дзеркально з ліва на право або з права на ліво.

- **Дублювання:** Дублювання копіює вишивку в обраному напрямку, збільшуючи розміри полотна.
